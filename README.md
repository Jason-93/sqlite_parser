# sqlite_parser

Parse SQLite files with hachoir


## Let's go

It's a python script and we can  run directly in the python3 environment  

### Download

Get code form github 

```
# git clone https://github.com/Jason-93/sqlite_parser.git
```

### Run the script


```
# cd sqlite_parser
# python parser.py  file 
```
